Industrial & Petrochemical

For energy-intensive industries, Wetown provides Non-Segregated Phase Bus (NSPB) and Cast Resin Busway (CRB) to ensure continuous operation and safety in hazardous areas.

  • Petrochemical: Sinopec & CNPC Projects - Using explosion-proof and corrosion-resistant CRB systems.
  • Data Centers: Huawei Cloud Global Command Center (Gui’an) - High-reliability low-voltage distribution for mission-critical servers.
  • Manufacturing: Inner Mongolia Chuanyuan Metal (Aluminium Smelter) - High-current rectification busbars for industrial processing.
