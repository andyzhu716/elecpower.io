# Non-Segregated Phase Bus (NSPB) Systems

Wetown provides non-segregated phase bus systems for power generation and industrial power distribution applications. NSPB is a practical enclosed bus solution for projects that require reliable electrical performance, compact structure, and flexible engineering design.

Compared with isolated phase bus systems used for larger generator outlet circuits, NSPB is often selected for auxiliary power connections, medium-voltage distribution, and other applications where space efficiency and system integration are important.

## What Is Non-Segregated Phase Bus?

Non-segregated phase bus, or **NSPB**, is a type of metal-enclosed bus system in which multiple phase conductors are installed within a shared enclosure space or in a less isolated arrangement than IPB.

It is widely used in power plants, substations, and industrial facilities where medium-voltage or auxiliary power distribution requires a compact and dependable busbar solution.

## Key Features

### Compact structure
NSPB offers a space-saving enclosed design that helps simplify routing and installation in electrical rooms, substations, and plant auxiliary systems.

### Reliable electrical insulation
The system is designed to maintain insulation performance under normal industrial operating conditions and can be adapted for different environmental requirements.

### Flexible engineering arrangement
Wetown can customize the bus layout, connection interfaces, support structures, and installation configuration based on the project design.

### Convenient system integration
NSPB can be integrated with switchgear, transformers, auxiliary systems, and other electrical equipment as part of a complete power distribution solution.

### Reduced maintenance complexity
The enclosed structure helps protect conductors from external contamination and improves organization compared with open conductor arrangements.

## Technical Design Considerations

NSPB design is typically based on:

- Rated voltage
- Rated current
- Short-circuit withstand requirements
- System frequency
- Installation environment
- Equipment interface dimensions
- Routing path and civil constraints
- Protection requirements
- Maintenance accessibility

Final system configuration should be determined according to project drawings, equipment arrangement, and electrical operating conditions.

## NSPB vs IPB

| Item | NSPB | IPB |
|---|---|---|
| Structure | Multiple phases in a shared / less isolated enclosure arrangement | Each phase in a separate enclosure |
| Typical Use | Auxiliary power and medium-voltage distribution | Generator outlet circuits and larger power plant connections |
| Space Requirement | More compact | Larger installation space |
| Typical Current Level | Medium to high, depending on design | High current applications |
| Shielding Effect | Good, depending on design | Stronger phase isolation and shielding |
| Common Project Type | Auxiliary systems, substations, industrial plants | Large power plant generator-transformer circuits |
